OpenCopilot by default comes with embedded frontend and the copilot Rest API. You can access frontend on the path `/ui`. However, if you want more customization you can do that with our open source repository. Instructions on how to build and modify: [frontend library](https://github.com/opencopilotdev/frontend-library).

You can also integrate the frontend separately by embedding it straight into your application.

1. Create an html file (for example index.html):

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>OpenCopilot UI</title>
    <script src="https://unpkg.com/opencopilot-ui@0.0.7/opencopilot.js"></script>
</head>
<body>
    <div id="embedded" style="width: 100vw; height: 100vh;"></div>
    <script>
        OpenCopilot.initialize({
            elementId: "embedded", // The argument is the id of the element where the component should be rendered
            theme: "light",
            isDebug: true,
            // authToken: "", // If auth is enabled in backend need to input JWT here
        });
    </script>
</body>
</html>
```

2. Open the created file in browser.
