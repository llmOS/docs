title: "Analytics"
---

If you wish to monitor your Copilot usage and have per user analytics, then you can do that with [Helicone](https://www.helicone.ai/).

To do so, you need to:

1. [Sign up](https://www.helicone.ai/signup) for Helicone and get an API key.

1. Add the key into the settings - setting the `helicone_api_key` variable. (TODO link to settings doc page)
